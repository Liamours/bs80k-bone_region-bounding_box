# Agents

Instructions for any coding agent working in this repo.

## Read first

- `intent.md` for the goal
- `context/` for background on the dataset and the matching approach

## Working rules

- Do not claim a method or number works unless it was run and checked against real samples
- Keep dataset facts to what is confirmed, do not guess counts or formats
- No AI co-author line on commits

## Skills

`.claude/skills/` holds reusable skill definitions for this repo, for example the bounding box from template matching skill. These are plain markdown, any agent can read them even without Claude specific tooling.
